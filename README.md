# Tally Counter API

## Käytetyt versiot

- Node.js: 18 (tai uudempi)
- Express: 4.18.2
- Winston: 3.11.0

## Asennus ja käyttö

1. `npm install`
2. `node src/main.js`

## Rajapinnat

- GET /counter-increase : Kasvattaa laskuria
- GET /counter-read : Lukee laskurin arvon
- GET /counter-reset : Nollaa laskurin
